----------------------------------------------------------------
- File: app/api/embedding_router.py
- The 4 embedding endpoints (POST/DELETE /api/embeddings/{pdf_id}/chunks,
  GET /api/embeddings/{pdf_id}/chunks, GET /api/embeddings/{pdf_id}/embedding-status)

  that all endpoints except /health require Authorization.

  signature requiring user_id).

    PDFService.get_by_id(pdf_id, session, current_user["user_id"]).

New Feature: PDF Text Extraction
---------------------------------
- File: app/services/pdf_service.py
  + Added `extract_text(doc_id, db, user_id)` static method:
    - Downloads PDF from Supabase Storage via `supabase.storage.download()`
    - Uses `PdfReader` (pypdf) to extract text page by page
    - Updates `page_count`, `extracted_text`, `status = EXTRACTED`
    - On failure: sets `status = FAILED` + `error_message`
- File: app/schemas/pdf_schema.py
  + Added `ExtractResponse` schema (id, page_count, char_count, status, message)
- File: app/api/pdf_router.py
  + Added `POST /api/pdf/{doc_id}/extract` endpoint (auth required)
